Viewport Units


vw // 1/100th viewport width
vh // 1/100th viewport height
vmin // 1/100th of the smallest side
vmax // 1/100th of the largest side

To Clarify

1vmax equals 1vh in portrait mode, whilst in landscape mode, 1vmax will equal 1vw.

Use case


.sidebar {
  height: 100%;
}
